# Спецификация Telegram-бота FindOrigin

## 1. Назначение программы

FindOrigin — это Telegram-бот, который получает текст или ссылку на пост и пытается найти первоисточник этой информации в интернете.

## 2. Стек

- **Next.js**
- **Деплой:** Vercel
- **Webhook**

### Требования к обработчику:
- Принимать POST-запросы от Telegram (update)
- Получать `chat.id` и `text` из `update.message`
- Отправлять ответ через Telegram API методом `sendMessage`
- Возвращать `200 OK` как можно быстрее (без долгих операций)

## 3. Рабочий процесс

### 3.1 Ввод данных

Пользователь:
- вводит текст, или
- присылает ссылку на Telegram-пост

### 3.2 Обработка данных

Программа:
- принимает текст напрямую, или
- извлекает текст из поста

Выделяет:
- ключевые утверждения
- даты
- числа
- имена
- ссылки

### 3.3 Поиск возможных источников

Ищет информацию в:
- официальных сайтах
- новостных сайтах
- блогах
- исследованиях

## 4. Анализ с помощью AI

С помощью AI бот сравнивает смысл, а не буквальное совпадение текста.

В результате выдает:
- список возможных источников (1–3 ссылки)
- оценку уверенности
